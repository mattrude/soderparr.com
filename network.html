The Soderholm Family Network

The Email Service

Email service is provided by G Suite by Google. This is the same service as gmail but uses the soderparr.com instead.

Instant Messaging

This service provides Extensible Messaging and Presence Protocol (XMPP), an open XML technology for real-time communication, which powers a wide range of applications including instant messaging, presence and collaboration. XMPP allows for communication between users on the same server and users across many servers at the same time.

How to use the service

This service requires the use of a client to connect to the server with. My personal preferred clients are Gajim or Swift for desktop/laptops and Conversations, Xabber or Freelab for mobile.

After installing the software, you should be prompted for your JID (username) and password, if it prompts you for a hostname or port, please see below. Your XMPP account is separate than any other account on the network, if you do not know your username and/or password, please contact Matt.

Connecting to XMPP Server

  • TCP Port: 5222
  • Domain: soderparr.com
  • Hostname: soderparr.com
  • Conferences: conference.soderparr.com

Security

  • SSL/TLS encryption is required between clients to servers (C2S) and server to server (S2S) connections
  • The backend is configured to store hashed and salted authentication data
  • Use Off-the-Record (OTR) or OMEMO in your chat client to have fully encrypted chats
